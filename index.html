          Documentação
          
Guia do Projeto

          
Tudo que você precisa saber para usar o SenkoLib.

        

        
      

      


        

          
📁 Estrutura de arquivos

          
O projeto é composto por:

          
    
            
  • index.html — interface principal
    • 
            
  • script.js — lógica da aplicação
    • 
            
  • styles.css — estilos
    • 
            
  • senkolib-core.js — motor de registro de layouts e variantes
    • 
            
  • layouts/layoutsXXX.js — arquivos com os layouts
    • 
            
  • variants/nome-do-layout.js — arquivos com variantes de cada layout
    • 
          

        


        

          
➕ Criar um novo layout

          
Clique em Adicionar no cabeçalho. Preencha os campos:

          
    
            
  • ID — identificador único, sempre em minúsculo. Ex: section-58. Será o nome
              do futuro arquivo de variantes.
    • 
            
  • Nome — nome de exibição. Ex: Section-58
    • 
            
  • Tags — palavras-chave separadas por vírgula para facilitar a busca
    • 
            
  • HTML / CSS — cole o código do layout
    • 
          

          
Copie o objeto gerado e cole dentro do array em layouts/layoutsXXX.js.

        


        

          
📦 Adicionar um novo arquivo de layouts

          
Crie um arquivo layouts/layouts002.js com a estrutura:

          
// @ts-nocheck
SenkoLib.register([ /* objetos aqui */ ]);

          
Depois adicione no index.html, antes de script.js:

          
<script src="layouts/layouts002.js"></script>

        


        

          
🔀 Criar uma variante de layout

          
Clique no botão + no card do layout desejado. Dentro da janela de variantes, clique em
            Nova Variante, preencha nome, HTML e CSS, e copie o objeto gerado.
          

          
Cole o objeto dentro do array no arquivo de variantes correspondente em variants/.

        


        

          
📄 Arquivo de variantes

          
O nome do arquivo deve ser o nome do layout em minúsculo. Ex: layout
            Section-58 → arquivo variants/section-58.js.
          

          
Estrutura do arquivo:

          
// @ts-nocheck
SenkoLib.registerVariant('section-58', [
  { nome: 'section-58.1',
    html: `...`,
    css:  `...` },
]);

          
Após criar o arquivo, adicione no index.html antes de script.js:

          
<script src="variants/section-58.js"></script>

        


        

          
🔍 Busca e visualização

          
Use a barra de pesquisa para filtrar por nome ou tag. Os botões de visualização abrem um preview completo
            com o código HTML e CSS do layout ou variante.
